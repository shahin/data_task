# TableTask

Provides a Rake task for managing table builds, analogous to Rake's built-in FileTask.

## Installation

Add this line to your application's Gemfile:

    gem 'tabletask'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install tabletask
